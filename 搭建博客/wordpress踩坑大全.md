# 无法登陆
突然有一天，我输入正确的账户和密码，但是死活都登录不上去，发送邮箱重置密码也失败，网上试了很多种方法包括：
- 清理浏览器缓存
- 看wp-config里的密码配置是否正确
- 禁用插件
- ...
但是都不管用，最终突然发现通过代码重新设置密码就好了：
![!\[\[Pasted image 20241102204629.png\]\]](assets/image15.png)
在主题里面找到有`functions.php`的这个文件，打开它，然后在一开始加上这句话：
![!\[\[Pasted image 20241102204754.png\]\]](assets/image16.png)
`wp_set_password( 'my_new_password', 1 );`
这里的1代表着用户的ID，你自己的用户一般来说第一个创建的ID都是1（如果不是的话，可以去数据库里面查一下）保存之后，然后在登录界面刷新一下，相当于执行了这个代码，密码成功修改，然后再把这一行删掉，然后再登录，就能成功登陆啦！
